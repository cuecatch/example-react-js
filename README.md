# CueCatch React.js Integration Example

CueCatch is a web component that allows you to easily create and embed surveys into your web applications. CueCatch surveys are purpose-built for the web and are designed to be embedded into any web application. This example demonstrates how to integrate the CueCatch web component into a React project.

## Get Started

This example project is built on React + TypeScript + Vite. To run it locally (http://localhost:5173/), follow the steps below:

```bash
npm install
npm run dev
```
